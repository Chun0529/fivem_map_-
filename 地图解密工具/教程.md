 **要求**
安装Python： 3.10-3.13
https://www.python.org/downloads/
安装好后 在地图文件夹里点击路径 安装下方依赖项(不懂查看1照片)
安装依赖项： `pip install requests colorama pycryptodome`.

 **使用**
别的文件不要动 双击simple_gui.py 第一次打开可能会让你选择打开方式 用Python打开
打开后填写要解密的服务器密钥 前往密钥资产下载你需要解密的文件 
将地图放入assets 文件夹。确保每个地图文件夹都包含一个 .fxap 文件
点击开始解锁 分享给你的好朋友享用吧
